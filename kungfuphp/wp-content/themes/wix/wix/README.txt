== Theme Description ==

Wix is a Twitter bootstrap based responsive WordPress theme with Masonry layout which can be used for personal or corporate websites.
Developed using Bootstrap 3 that makes it mobile and tablets friendly. Wix incorporates latest web standards such as HTML5 and CSS3 and is SEO friendly.
It has Theme Options to manage logo, favicon and footer text.

== Features ==

Responsive design(Bootstrap), Fluid Layout (Masonry) ,Theme Options, Custom Widgets, Custom Menu, Left-sidebar, Right-Sidebar template, Full-width template, blog template.

== Installation ==

1. Primary:

 = Login to your wp-admin account and go to Appearance -> Themes.

 = Select "Install" tab and click on the "Upload" link.

 = Select "theme .zip" and click on "Install Now" button.

 = In case of errors, use an alternate method.



2. Alternate:

 = Unzip the template file (wix.zip) that you have downloaded.

 = Upload the entire folder (wix) to your server via FTP and place it in the /wp-content/themes/ folder.

 = Do not change the directory name.

 = The template files should be there now: /wp-content/themes/wix/index.php (example).

3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "Wix" to activate the theme.

5. Complete all of the required inputs in the theme Options page (in the WP admin panel) and click "Save".



6. Change Logo and Favicon:

 = Login to your wp-admin area and go to Appearance -> Theme Options.

 = Select "Basic Settings" Tab.

 = Upload the logo, that you require & Enter Introduction text for right of Your Theme.
 
 
7. Wix Author information :

 = Log into your WP admin panel and click on "Appearance". Go to "Widget" tab.
 
 = Drag widget in sidebar and set widget title, content, image and set social links.
 
8. posts comments and Tags :
 
 = Drag widget in sidebar and posts comments and Tags are coming in tabs.
